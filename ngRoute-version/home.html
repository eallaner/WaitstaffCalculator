Welcome to Waitstaff Calculator   




This is a tip calculator for waiters/waitress during a shift.


To use it, please go to “New Meal“, enter the base meal price, tax percentage and tip percentage to calculate. If you want to clear up the input areas, please click the Cancel button.You can check out each meal charge on the same page.


To check out the summay of your earnings, please go to “My Earnings “. If you want to reset the whole shift earnings, please click the Reset button.


Enjoy!
